# Solr Folder

To start the Solr service, run:
```
bin/solr start
```

## How to start the application

Step 1. Navigate to the `backend` folder and run:
```
npm install
node index.js
```

Step 2. In this directory, run:
```
bin/solr start
```

Step 3. Navigate to `frontend` folder and run:
```
npm install
npm start
```

Step 4. Navigate to `localhost:3000` to access the application.
